# Design Spot — Portfolio (static)

This is a lightweight, responsive static portfolio for the graphic design agency "Design Spot".

Theme color: DEEP NAVY #0B132B

What's included:
- `index.html` — main site
- `css/styles.css` — styles
- `js/script.js` — small interaction helpers
- `assets/` — SVG placeholders (logo, thumbnails, hero art)

How to run:
1. Open `index.html` in a browser (double-click or serve via a static server).
